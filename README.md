# description
ニコ生の番組視聴ページにサイドバーを設置します。  
  
サイドバーには、フォロー中の番組がライブサムネイル付きで表示されます。  
内容は定期的に自動更新されます。  
  
番組を視聴しながら別の番組を監視したい時に便利です。  
「別窓くん」と併用するとさらに便利になります。
